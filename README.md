# AutoReception

Clean marketing and demo site for AutoReception — a missed-call recovery and SMS lead-warming offer for plumbers and home-service businesses.

## Positioning

AutoReception is not generic lead generation. It focuses on the leak that happens after marketing works:

1. A homeowner calls.
2. The business misses the call.
3. AutoReception sends a fast, transparent text-back.
4. The lead is warmed and qualified.
5. The owner gets a callback-ready job card.
6. Weekly reporting shows recovered opportunities and estimated revenue.

## Routes

- `/` — focused homepage
- `/demo` — interactive missed-call recovery demo with local mock data
- `/privacy` — launch-baseline privacy policy with SMS consent language
- `/terms` — launch-baseline terms
- `/robots.txt` and `/sitemap.xml` — generated by Next.js metadata routes

## Development

```bash
npm install
npm run dev
npm run lint
npm run build
```

The app intentionally does not require Supabase, Stripe, Twilio, OpenAI, or other secrets to build. Demo content is illustrative until real customer integrations are configured.
